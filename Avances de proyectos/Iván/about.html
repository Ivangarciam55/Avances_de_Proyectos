Acerca del Proyecto

¿Qué es WeatherApp?

WeatherApp es una aplicación web que proporciona información meteorológica en tiempo real. A través de la API de OpenWeatherMap, los usuarios pueden obtener datos detallados sobre el clima en cualquier ciudad del mundo.

Objetivos del Proyecto

  • Brindar información meteorológica precisa y actualizada.
  • Proveer una interfaz intuitiva y fácil de usar.
  • Asegurar que la aplicación sea accesible desde dispositivos móviles y de escritorio.

Tecnologías Utilizadas

  • HTML5
  • CSS3
  • JavaScript
  • API OpenWeatherMap

Créditos

Los datos meteorológicos son proporcionados por la API de OpenWeatherMap. El diseño y desarrollo de la aplicación fue realizado por nuestro equipo.
